# AI Agents in LangGraph

A Python project for building intelligent agents using LangGraph and OpenAI's GPT models.

LangGraph helps describe and orchestrate the control flow. It is an extension of LangChain that supports graphs. 
Single and Multi-agent flows are described and represented as graphs, allowing for extremely CONTROLLED "flows". 
Built-in persistence allows for human-in-the-loop workflows

## Features

- Build autonomous agents with LangGraph
- Integrate with OpenAI's Chat API (GPT-3.5-turbo, GPT-4)
- React pattern for agent reasoning
- Easy setup with virtual environments

## Installation

```bash
# Clone or navigate to project
cd "AI Agents in LangGraph"

# Create virtual environment
python -m venv .venv
source .venv/Scripts/activate  # Windows: .venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
